# Голосовой ассистент на базе ИИ

Виртуальный помощник с голосовым управлением, использующий современные технологии машинного обучения.

## Возможности

- Голосовой ввод (Google Speech Recognition)
- Голосовой вывод (pyttsx3)
- ИИ-ассистент через Hugging Face
- Управление приложениями Windows
- Погода, время, системные команды
- Графический интерфейс на PyQt5
- Ручной ввод текста

## Установка

```bash
git clone https://github.com/s1maqq/desktop-voice-assistant
cd desktop-voice-assistant
pip install -r requirements.txt
python main.py
